Spotify: Case Study

Completed a UX Case Study

This case study delves into solving a user problem for an already launched product, Spotify. Given a user persona and user journey map, another developer and I were challenged with rethinking the information architecture of the music platform, wireframing potential solutions to eliminate the users pain points while using the application, and creating prototypes of new UI design ideas.
